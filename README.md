# 神秘塔罗占卜工具

现代年轻人审美风格的塔罗牌占卜网页应用。

## 功能特点

- 🎨 现代霓虹设计风格
- 🔮 多种占卜类型（单张牌、三张牌、凯尔特十字等）
- ✨ 动态卡片效果和动画
- 📱 响应式设计，支持移动设备
- 🌙 多种视觉主题切换
- 📊 能量等级分析和详细解读

## 本地运行

1. 下载项目文件
2. 打开 `index.html` 文件
3. 在浏览器中开始使用

## 部署方法

### 方法一：GitHub Pages（免费）
1. 在GitHub创建新仓库
2. 上传项目文件
3. 进入仓库设置 → Pages
4. 选择主分支，保存

### 方法二：Vercel（免费）
1. 注册Vercel账号
2. 导入GitHub仓库
3. 一键部署

### 方法三：Netlify（免费）
1. 注册Netlify账号
2. 拖拽项目文件夹到部署区域
3. 自动生成在线链接

## 技术说明

- 纯HTML/CSS/JavaScript实现
- 无需后端服务器
- 所有数据存储在本地
- 使用Font Awesome图标库

## 使用说明

1. 选择占卜类型
2. 输入问题焦点（可选）
3. 选择牌阵方向和主题
4. 点击"洗牌并抽牌"
5. 查看详细的塔罗牌解读
